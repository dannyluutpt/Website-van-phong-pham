# Mực Tím — Website văn phòng phẩm 🎒✏️

Website mẫu bán văn phòng phẩm cho học sinh, được xây dựng để **quảng bá dịch vụ thiết kế website**.

Phong cách: hồn nhiên, tươi tắn, nhiều màu sắc — áp dụng các xu hướng thiết kế web mới nhất (bento grid, gradient sống động, micro-interactions, scroll animations).

## ✨ Tính năng

- **Hero động** với hình minh hoạ SVG, sticker bay lượn theo chuột (parallax).
- **Bento grid** danh mục sản phẩm.
- **Lưới sản phẩm** có bộ lọc theo danh mục, nút yêu thích, thêm vào giỏ.
- **Giỏ hàng** trượt (drawer) đầy đủ: tăng/giảm số lượng, xoá, tính tổng.
- **Flash sale** với đồng hồ đếm ngược thời gian thực.
- **Animation cuộn trang** (reveal), bộ đếm số liệu, thanh tiến trình cuộn.
- **Marquee** chạy chữ, toast thông báo, nút lên đầu trang.
- Responsive đầy đủ cho mobile, hỗ trợ `prefers-reduced-motion`.

## 🛠️ Công nghệ

HTML + CSS + JavaScript thuần — không cần build step, deploy tĩnh ở bất kỳ đâu.

## 🚀 Chạy thử

Mở trực tiếp `index.html` trong trình duyệt, hoặc chạy server tĩnh:

```bash
python3 -m http.server 8000
# mở http://localhost:8000
```
